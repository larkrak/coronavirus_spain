Landing

You are able to chose between entering Spain, World, or Covid information. The option you are about to select will increase to show clearly which one will be shown.

Spain

  1. Map delimited by provinces. Colors by CCAA
  2. Summary of total cases, hospitalization, uci, and death
  3. Graph when you can see the progression of each type of data.
  4. Graph when you can see the comparison between a province and his autonomous community
  5. Controls where you can specify a new type of data to be shown and the days it will show.
  6. Control where you can add the rest of the regions, if inside that autonomous community has more provinces.

Restrictions

Section where you can see the restrictions of that province

World

  1. Select a country, this changes the values of the boxes (2)
  2. Info boxes, here you can see the current information of the selected country, the valuew in the center is today's value. and the one below is the total
  3. It is a progress bar that shows the percentage of cases, recover or death based on the population of the country.
  4. This player controls the data of which the map(6) and table(8) use to show information.
  5. Displays the date of the data that is being shown
  6. Circules that are linked to a country, the bigger the circule the more cases there are.
  7. Information box (pop-up), when you click on a circule it will display the information of the country
  8. Ranking Table, it changes when you click on a different type of search (2)
